agent: telamon

Work on the following epic, create a folder and backlog for the epic in `.ai/telamon/memory/work/active/<epic-folder>/backlog.md`, 
follow the `telamon.plan_story` skill for each of the stories in the epic backlog, 
creating a `.ai/telamon/memory/work/active/<epic-folder>/<story-folder>/backlog.md` for each story, 
and only after all of them are planned follow the `telamon.implement_story` skill for each of them: $1

When you need to create a temporary file, use the `telamon.thinking` skill.
